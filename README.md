# pickpackおべんとう 🍱
食欲の秋、行楽の秋、運動会。
好きなおかずや秋の味覚、彩りを詰め込んで、外へ出かけたくなる季節。

「今日はどんなお弁当を食べようかな？」

RUNTEQ ミニアプリWeek 2025にて、
* グッドコンセプト賞
* オーディエンス賞　を受賞しました🏆

# 📋 概要
pick（選ぶ）pack（詰める）とパクパクを掛け合わせました。
今の気分や好みにあわせて、自分だけのお弁当を作ることができるアプリです。
秋の味覚を楽しみながら食材を選び、食べる、ごちそうさまを疑似体験できます。

# ✨ 特徴と工夫した点
1. 秋を感じる豊富なラインナップ
美味しそうと感じられるあたたかみのあるイラストを採用し、さつまいもや栗など秋の味覚をふんだんに取り入れました。
彩りやバランスを考えながら、楽しく食材を選べるように設計しています。

2. オリジナリティの追求
完成したお弁当には、自分でお好みの名前をつけることができ、
さらにSNSで作ったお弁当をシェアすることができます。

3. 「いただきます」から「ごちそうさま」まで
お弁当を作って終わりではなく、「ぱくぱくもぐもぐボタン」を実装しました。

# 🛠 使用技術

# 📅 開発背景
RUNTEQのイベント「ミニアプリWeek 2025」（2025/09/12 〜 09/21）にて制作しました。
テーマが『秋』だったため、食材や背景で季節感を取り入れています。
限られた期間の中で、ユーザーが好みの食材を詰め込むときめきと、
ほっと一息ついて心がほぐれるようなミニアプリを目指しました。
